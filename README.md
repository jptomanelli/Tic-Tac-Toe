# Tic-Tac-Toe

> A Tic-Tac-Toe Program for TTP

## Build Setup

``` bash
# Clone project
git clone https://github.com/jptomanelli/Tic-Tac-Toe

cd Tic-Tac-Toe

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
